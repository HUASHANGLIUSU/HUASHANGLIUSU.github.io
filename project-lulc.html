LULC 校园土地覆盖分类与精度验证

以华东师范大学闵行校区为研究区，基于 10 m 四波段遥感影像完成最大似然监督分类，并使用 Python 对分类结果进行对象级空间规则后处理，最终以实地照片样点进行四类精度验证。

PythonENVIRasterioNumPyscikit-imageGIS

项目概览

项目形成了“ENVI 最大似然分类 + Python 对象级空间规则后处理 + 地面样点精度评价”的完整流程。分类阶段保留 river、grass、tree、building、runway、fakegrass 和 road 七个精细类别；精度评价时统一归并为 Water、TreeCover、Grassland 和 Impervious / Artificial Surface 四类，以便与 SinoLC-1 进行同点位比较。

数据与研究区

  • 研究区：华东师范大学闵行校区
  • 影像：B02、B03、B04、B08 四波段
  • 空间分辨率：10 m
  • 输出网格：139 × 163 像元
  • 坐标系：WGS 84 / UTM Zone 51N

分类方法

  1. 在 ENVI 中构建七类训练 ROI
  2. 运行 Maximum Likelihood Classification
  3. 计算 NDVI、NDWI 与局部纹理
  4. 按类别实施连通性、形状和邻域约束
  5. 保存 FinalV3 并开展精度评价

Python 空间后处理

Python 部分属于规则驱动的对象级空间后处理，并非重新训练深度学习模型。处理使用 Rasterio、NumPy、SciPy 与 scikit-image，针对水体连通、跑道环状结构、树草混淆、住宅区人工地表以及孤立小斑块分别制定保守修正规则。

地面验证

主评价使用 35 个有效评价样点：Water 3 个、TreeCover 13 个、Grassland 8 个、Impervious 11 个。所有点均来自校园实地照片及水印坐标，同一批点分别用于 SinoLC-1 和 FinalV3 的中心像元评价。

精度结果

SinoLC-1 的 OA 为 31.43%，Kappa 为 0.0071；FinalV3 的 OA 为 51.43%，Kappa 为 0.3246。FinalV3 在本次样本集上表现更好，但总体精度仍有限，复杂校园地物间仍存在明显混淆。

主要混淆

  • TreeCover → Impervious：6 个样点
  • Impervious → Grassland：4 个样点
  • TreeCover → Grassland：3 个样点
  • Grassland → TreeCover：2 个样点

结论与局限

该项目验证了空间先验能够改善像元级分类中的明显碎斑和不合理结构，但图面连续性改善并不等同于精度必然提高。10 m 混合像元、建筑与道路光谱相似、树草边界以及样点数量和空间分布仍限制结果。后续可结合更高分辨率影像、更均衡的独立样点和语义分割模型继续优化。
